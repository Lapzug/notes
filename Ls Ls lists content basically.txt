bash scripting
1. Creating a Bash Script
you can make bash script by using .sh and nano

open terminal and use nano
bash

nano myscript.sh
Add the following basic script structure to the file:

bash

#!/bin/bash


echo "Hello, World!"  
 #!/bin/bash is called a "shebang," uses a ball shell to run script
The echo command prints the text to the terminal.
Save and close the file (in nano, you can press Ctrl + O to save and Ctrl + X to exit).

2. Making the Script Executable
Before you can run your script, you need to make it executable:

bash
Copy code
chmod +x myscript.sh
3. Running the Script
Now you can run your script by typing:

bash
Copy code
./myscript.sh
You should see:

Copy code
Hello, World!
